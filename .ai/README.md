# korrektsiya-zreniya.github.io

## Описание проекта

Независимый социальный проект, информирующий о рисках и последствиях лазерной коррекции зрения (LASIK и др.). Сайт публикует реальные истории пациентов, научные данные и судебные дела, чтобы помочь людям принять осознанное решение.

**URL**: https://korrektsiya-zreniya.net

## Технологический стек

| Компонент | Технология                      |
| --------- | ------------------------------- |
| SSG       | [Hugo](https://gohugo.io/)      |
| Тема      | Mainroad                        |
| Хостинг   | GitHub Pages                    |

## Структура проекта

```
├── archetypes/          # Шаблоны для нового контента
├── assets/              # CSS, изображения для обработки Hugo
├── content/             # Все статьи сайта (Markdown)
│   ├── novosti/         # Новости (9 статей)
│   ├── oslozhneniya/    # Осложнения после ЛКЗ (27 статей)
│   ├── otzyvy/          # Отзывы пациентов (8 статей)
│   ├── pobochnye-effekty/ # Побочные эффекты (8 статей)
│   ├── riski-i-posledstviya/ # Риски и последствия (19 статей)
│   └── sudebnye-dela/   # Судебные дела (3 статьи)
├── layouts/             # Кастомные шаблоны и shortcodes
│   ├── partials/        # Частичные шаблоны
│   └── shortcodes/      # Hugo shortcodes (card, cards, img)
├── static/              # Статические файлы (favicon, manifest)
├── themes/mainroad/     # Git submodule темы
├── hugo.toml            # Главный конфиг Hugo
└── public/              # Сгенерированный сайт (не редактировать)
```

## Задачи проекта

1. **Информирование** — Предоставление объективной информации о рисках ЛКЗ
2. **Документирование** — Сбор реальных историй пострадавших пациентов
3. **Правовая поддержка** — Информация о судебных делах и правах пациентов
4. **Сообщество** — Телеграм-чат пациентов: [@lasik_chat](https://t.me/lasik_chat)

## Локальная разработка

```bash
# Установка Hugo (macOS)
brew install hugo

# Клонирование с подмодулями
git clone --recursive https://github.com/korrektsiya-zreniya/korrektsiya-zreniya.github.io.git

# Запуск dev-сервера
hugo server -D

# Сборка для продакшена
hugo --minify
```

## Добавление контента

### Новая статья

```bash
hugo new novosti/my-article.md
```

description: "Краткое описание для SEO"

## Деплой

Сайт автоматически деплоится через GitHub Actions при пуше в `main` ветку.

## Конфигурация (hugo.toml)

- **Тема**: Mainroad с боковой панелью справа
- **Виджеты**: поиск, теги, последние статьи
- **SEO**: OpenGraph, Schema.org включены
- **Дата**: формат DD.MM.YYYY (русский)
